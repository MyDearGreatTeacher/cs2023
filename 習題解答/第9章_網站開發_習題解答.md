# 第9章 網站開發

- 選擇題  BDBDD ABADA BCABB DBCAC

## 簡答題
```
1. 請列出本章內容所說用來開發網站的3種技術，並簡單定義之。
答：常用來開發網站的3種技術如下：
•HTML：用來描述網頁的內容，包括標題、段落、影像和連結。
•CSS：設定網頁內容的外觀，如字型、色彩、邊界、背景等。
•JavaScript：用來指定網頁的行為，或是將內容以動態方式顯示提升互動性。

2. 請列出使用網站建構軟體製作網站的步驟。
答：使用網站建構軟體製作網站的步驟如下：
1. 在軟體廠商的網站上註冊申請服務。
2. 替你的網站挑選某種模板或設計。
3. 選擇網站的網域名稱。
4. 設定SEO和其他網站選項。
5. 設計網站並使用拖放介面和視覺化編輯器輸入網站內容。

3. 請列出至少3個在選擇網站寄存方案時，應該考慮的重要因素。
答：在選擇網站寄存方案時，有些重要因素必須考慮：（任答3個）
•決定需要的磁碟空間大小
•決定需要的頻寬大小
•確定作業系統與伺服器上的軟體相容
•檢查主機的可靠性

4. 請問網站分析工具能提供什麼樣的分析資料？請列出至少4個。
答：分析資料可能涵蓋：（任答4個）
•線上的訪客人數
•訪客來自哪裡
•最多人瀏覽的網頁是哪些
•訪客在每個頁面的停留時間
•訪客使用的裝置和瀏覽器
•訪客走過什麼路徑來探索內容
•新到訪和回訪的人數
•訪客瀏覽多少個頁面
•統計某一段時間的訪客人數

5. 請列出建構與發布網站過程的步驟。
答：以下是建構與發布網站過程的步驟：
1. 安排和設定網頁寄存服務（假如不是使用學校提供的網站伺服器）。
2. 設計網站（版片配置、導覽、內容）。
3. 創作或取得圖片和多媒體資源。假如是大型影像檔案，事先編輯檔案大小以利快速下載。
4. 撰寫每個網頁的程式碼。
5. 在多種瀏覽器和不同大小螢幕的裝置上測試預覽網站，確認網站在上面執行的外觀和行為都正確。
6. 將更新過的網站檔案以FTP方式發布到伺服器上。
7. 在一或多種瀏覽器上測試觀看網站新發布的版本，確認所有檔案都已正確更新。
8. 假如在網站上新增內容，就要重複執行步驟4到7。
```
